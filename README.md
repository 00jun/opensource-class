# 오픈소스 개발 방법론

 * 오픈소스를 알고 잘 사용할 수 있도록 한다.
 * 오픈소스 프로젝트들에 여러 형태로 기여하고 참여할 수 있는 방법과 그에 필요한 여러가지 기술들을 소개하여 학생들의 기술 성장을 돕는다.
